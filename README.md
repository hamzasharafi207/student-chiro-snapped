Student Chiro Snapped is a concept web platform designed to help student athletes discover chiropractors who understand sports injuries, training demands, and student-friendly pricing.

This project was built as a portfolio demonstration of backend routing, template rendering, and responsive front-end design using Flask.

Overview

The application allows users to:

Browse a list of chiropractors

Filter by city, sport focus, student-friendly pricing, and evening hours

View detailed chiropractor profiles

Submit new chiropractor entries (demo preview flow)

This is a demo application and does not store real healthcare data.

Tech Stack

Python

Flask

HTML (Jinja templating)

CSS (custom responsive styling)

Key Features

Template inheritance using base layout

Dynamic URL routing with Flask

GET-based filtering system

Multi-page navigation

Profile detail pages

Demo submission and preview flow

Responsive UI design

Project Structure

app.py
Handles routing, filtering logic, and request processing.

templates/
Contains all HTML templates using Jinja templating.

static/styles.css
Custom CSS styling and responsive layout.

How to Run

Install Flask:

pip install flask


Run:

python app.py


Open your browser and visit:

http://127.0.0.1:5000
